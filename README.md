# [NOMBRE]

## Descripción

En esta actividad desarrollarás una solución en Python utilizando **Python 3.12** dentro de **GitHub Codespaces**. El ejercicio consiste en implementar la lógica requerida, probar su funcionamiento y documentar los resultados obtenidos.

## Objetivo

* Practicar el uso de Python 3.12.
* Desarrollar una solución funcional para el problema planteado.
* Ejecutar y validar el programa en GitHub Codespaces.
* Utilizar Git y GitHub para gestionar y entregar el trabajo.

## Cómo abrir el Codespace

1. Accede al repositorio de GitHub de la actividad.
2. Haz clic en el botón **Code**.
3. Selecciona la pestaña **Codespaces**.
4. Haz clic en **Create codespace on main** (o sobre la rama indicada por el docente).
5. Espera a que el entorno termine de configurarse.

Una vez abierto el Codespace, verifica la versión de Python:

```bash
python --version
```

La salida esperada debe ser similar a:

```text
Python 3.12.x
```

## Cómo ejecutar

1. Abre la terminal integrada de Codespaces.
2. Ubícate en la carpeta del proyecto si es necesario:

```bash
cd ruta/del/proyecto
```

3. Ejecuta el programa principal:

```bash
python main.py
```

4. Revisa la salida mostrada en la terminal.

Si la actividad incluye archivos adicionales o parámetros, utiliza el comando correspondiente. Por ejemplo:

```bash
python main.py archivo_entrada.txt
```

## Entregable

El alumno deberá entregar:

1. El código fuente completo de la solución.
2. Los archivos adicionales requeridos por la actividad.
3. Un commit final con los cambios realizados.
4. El repositorio actualizado en GitHub.

Antes de entregar, verifica que:

* El programa se ejecuta sin errores.
* El código está correctamente organizado.
* Los cambios han sido guardados y enviados al repositorio.

```bash
git add .
git commit -m "Entrega de la actividad [NOMBRE]"
git push
```
